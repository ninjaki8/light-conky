# Light Conky Theme

OS: Ubuntu 23.04
Conky: 1.18.0
Resolution: 2k (2560x1440)

## Installation

Use the package manager to install conky.

```bash
sudo apt install conky-all
```
Copy files to /home/user/.config/conky/

*Note: Create directory path if it does not exist

## Usage

```bash
conky

```
